HTML - Linguagem de Marcação de Hipertexto

O QUE É HTML?

HTML, ou Linguagem de Marcação de Hipertexto, é a linguagem padrão usada para criar a estrutura de uma página web, definindo o conteúdo e sua organização com elementos como títulos, parágrafos, imagens e links.

PARA QUE SERVE O HTML?

- Criar uma estrutura visual de um site;
- Permitir a navegação e interação por meio de links e botões;
- Facilitar que mecanismos de negócios indexem o conteúdo;
- Melhorar a acessibilidade para pessoas com deficiências, por exemplo, usando textos alternativos para imagens;
- Servir como base para aplicar outras tecnologias web, como CSS e JavaScript.

QUAL A DIFERENÇA ENTRE HTML E LINGUAGENS DE PROGRAMAÇÃO?

O HTML serve para estruturar e exibir conteúdo, enquanto linguagens de programação são usadas para criar programas com lógica e comportamentos dinâmicos. Portanto, HTML não é considerada uma linguagem de programação, mas sim uma linguagem de marcação essencial para construir páginas web.

QUAIS AS FUNÇÕES DAS TAGS:

  • !DOCTYPE html
      É uma declaração que informa ao navegador o tipo de documento, nesse caso um HTML.
  • html lang="en"
      A tag html é a raiz do documento HTML. O atributo lang="en" especifica o idioma do conteúdo da página, neste caso, inglês.
  • head
      A tag head contém metadados sobre o documento, como título, links para arquivos CSS, scripts, etc.
  • meta charset="UTF-8"
      Define a codificação de caracteres do documento como UTF-8, que suporta a maioria dos caracteres usados na web.
  • meta name="viewport" content="width=device-width, initial-scale=1.0"
      Esta tag é crucial para o design responsivo. Ela define a largura da viewport para a largura do dispositivo e o nível de zoom inicial como 1.0.
  • title
      A tag title define o título da página, que aparece na aba do navegador.
  • body
      A tag body contém todo o conteúdo visível da página web, como texto, imagens, links, etc.
  • h1/strong>
      A tag h1 serve para definir o título principal de uma página ou seção.
  • p
      A tag p é utilizada para definir um parágrafo de texto.
  • br
      A tag br serve para inserir uma quebra de linha dentro do texto, sem criar um novo parágrafo.
